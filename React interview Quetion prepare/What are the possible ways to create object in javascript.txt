    2)   Object's ceate method
        
        The create method of Object creates a new object by passeing
    the prototype object as a parameter

    3)  Object literal syntax

     The object syntax (or object intiaizer) is a comma, separated set of
      name-value pairs wrapped in curly braces
        var object = {
             name: 'sudheer',
	     age: 34,
	};
   
      Object literal property values can be of any data type , including
      array, function . and nested object

     Note: this is an easiest way to create to object

    4)  Function constructor
